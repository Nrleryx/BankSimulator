# Bank Account Simulator

A simple bank account management system written in C++.

## Features

- Deposit and withdraw money
- Check balance
- Different account types (Checking, Savings)
- File operations for saving/loading data

## Compilation

```bash
g++ -o banka Account.cpp VadesizHesap.cpp VadeliHesap.cpp BankaSimulator.cpp
```

## Running

```bash
./banka
```

## Usage

The program works with a menu-based interface. You can select the desired operation from the menu.
